# Icons

The icon system is a reexport of the [Heroicons](https://heroicons.com/) library. The icons are available in two variants: `outline` and `solid`. The icons are available in a variety of sizes and can be used in any context where an SVG can be used. On top of the Heroicons, there is also a reexport of SVG spinners from [Iconify.Design](https://icon-sets.iconify.design/svg-spinners/).
